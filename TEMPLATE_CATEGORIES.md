## 🎯 How to Use

Users can select templates by:
1. **Career Field** (Tech, Sales, Marketing, etc.)
2. **Job Title** (Software Engineer, Sales Executive, etc.)
3. **Current Role/Level** (Entry-level, Manager, Executive)

Each template includes:
- Suggested professions
- Detailed description
- Professional color scheme
- Role-specific sections
